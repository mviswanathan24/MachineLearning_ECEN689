Hint: the SVM loss function is not strictly speaking differentiable

$\color{blue}{\textit Your Answer:}$ since the svm loss function is a max function, at certain points it may not be differentiable because of the nature of the max function. Hence when we calculate gradient at some points, it fails. It is not something to worry about though, since this can be avoided by using few points while calculating gradient (implies lesser number of discontinuities), or you could calculate subgradients.
